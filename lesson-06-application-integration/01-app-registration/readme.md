> **Remember:**
> - Every app needs a **Client ID** (Application ID)
> - **Redirect URIs** must exactly match what the app sends
> - **Client secrets** expire and must be rotated
> - **Public clients** (SPA, mobile) use PKCE, not secrets

> - APIs **expose scopes**, client apps **request permissions**
> - `openid` and `offline_access` are common default scopes
